Description

Loss of power to real-time clock. Time has been set back

Cause

Action

Reset date and clock with the monitor-command "date" after the exchange of the battery. Otherwise the error- message 2.13.501 will appear with starting the controller. Syntax: date [yyyy-mm-dd] [hh:mm:ss], e.g.
