Description

Invalid travel mode in the case of travel key from MMI

Cause

An invalid or unsupported travel mode was requested by the HMI via one of the jog keys of the virtual keyboard. Possible reasons: The travel mode is not supported by the currently used firmware version Error in the HMI or during data transmission

Action

Make sure that a current controller firmware is used.All supported travel modes can be found in the documentation PC Interface for EEXC Controllers in chapter Assignment of the Jog Keys.

Effect
