Back up/Restore Vulcan system data

  1. Start HMI. Back up Vulcan system

  2. Select the “wrench” icon.

  3. Select “Save backup”. .Fig. 5‑7

    |

  1. Select the data to be saved.

    • Data: Parameter setting for the machine.

    • Log: Alarm log.

    • Programs: Customer and default programs. .Fig. 5‑8

      |

  1. The backup is done.

    • The backup file can be found by the tip and copied to other disk. .Fig. 5‑9

      |

  1. Select “Load backup” in HMI. Restore Vulcan system

  2. Select the backup file according to the date. .Fig. 5‑10

    |

  1. Press the check mark.

Back up/Restore Inovance drive data

Means, Tools, Materials * USB to RS232 converter cable, mat. no. 385230. * RS232 to RJ45 converter cable, mat. no. 359401.

  1. Connect your laptop to the respective X1/X2/Y/Z axis drive with communication cable by the port “CN5”. Connect InoDriveShop to Inovance drive .Fig. 5‑11

    |

  1. Check the COM port of the communication cable in device manager on your laptop. .Fig. 5‑12

    |

  1. Run software “InoDriveShop” on your laptop.

  2. Select “New project”.

  3. Choose “SV660N Series” and select the corresponding COMport as checked in device manager.

  4. Click “Next page”. .Fig. 5‑13

    |

  1. Click “Begin Scan” and wait unit the connected drive module being found.

  2. Put a check mark on the target drive module and click “Finish”. .Fig. 5‑14

    |

  1. Double click on “Multi-machine parameter management”. Back up Inovance drive

  2. Select “Export to recipe” and confirm query with “Yes”. .Export to recipe Fig. 5‑15

    |

  1. Wait unit the transmission is done.

  2. Acknowledge the error message by clicking “OK”. .Fig. 5‑16

    |

  1. Name the backup data and save it on your laptop.

  2. Go to “InoDriverShop” home page. Restore Inovance drive

  3. Double click “Multi-machine parameter management”.

  4. Select “Import recipe file”. .Fig. 5‑17

    |

  1. Select the corresponding Inovance drive data. .Fig. 5‑18

    |

  1. Confirm with “Yes”.

  2. Acknowledge the pop-up window about restoring parameter by clicking “OK”.

    • It’s normal that some system parameters or specific parameters fail to be written. .Fig. 5‑19

      |

  1. Import parameters successfully.

    • In case that some parameters are unexpectedly lost, please double check the following parameters.

  2. In “Work Space”, select “SV660N” and double click on “Param List”. .Fig. 5‑20

    |

  1. Check “H08” > “H08-19” (Speed feedforward gain). Setting value: 100. .Fig. 5‑21

    |

    • If the value is 100, the following step 23 to 28 can be skipped.

  2. Find the parameter “H9-00”, change the setting value to1 (enabled). .Fig. 5‑22

    |

  1. Right click on the item just modified and choose “Download the selected items” to save the parameters. .Fig. 5‑23

    |

  1. Set “H08-19” (Speed feedforward gain) to 100.

  2. Right click on it to download the selected items.

  3. Set “H09-00” back to “0” (Disabled).

  4. Right click on it to download the selected items.

  5. Check “H08” > “H08-00” (Speed loop gain).

    • X1/X2 drive: 140.

    • Y/Z drive: 170.

  6. If not, set the “H08-00” and right click on it to download the selected items.

  7. Check “H07” > “H07-26” (Flux weakening selection). Setting value: 1. .Fig. 5‑24

    |

  1. If not, change it to 1.

  2. Right click on the parameter to choose “Download the selected items”.

  3. Switch the main switch off and then switch the machine on again.

Back up/Restore the whole system

  1. Go to the machine startup interface. Back up the whole system

  2. Select “Backup & Restore”.

    • The startup interface will appear every time the machine is switched on or restarted. .Fig. 5‑25

      |

  1. Press “Create factory backup”. .Fig. 5‑26

    |

  1. Confirm that backup is finished successfully.

    • The backup can be exported to the U disk by “Export factory backup”.

  2. Restart or shut down the machine.

  3. Enter the “Backup and Restore” interface. Restore the whole system

  4. Select the backup to be restored.

  5. Press the “Restore factory backup” button (example).

  6. The restoration is done.
