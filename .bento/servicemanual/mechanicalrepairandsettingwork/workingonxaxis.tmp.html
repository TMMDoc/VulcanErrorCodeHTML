Roughly setting the X axis angle

The machine’s main switch is switched off and secured against being switched on again.

  1. Remove the side enclosure.

  2. Remove the X bellows front and rear.

  3. Release all the spring assemblies on both X1 and X2 axes.

  4. Undo both fixed stops (left and right) and push as far as possible in the X- direction.

  5. Pull motion unit by hand onto the two fixed stops.

  6. Tighten all the spring assemblies to torque:

    • Left: 40 Nm.

    • Right: 15 Nm.

  7. Do this additional work:

    • Set the fine angularity of the motion unit (see “Setting thefine angularity of the motion unit”, pg. xxx).

    • Determine the X axis reference point offset value (see “ Checking and setting the X axis reference point offset va-lue”, pg. xxx).

    • Set X axis limit switch (see “Setting X axis limit switch”, pg. xxx).

    • Set X axis fixed stop (see “Setting X axis fixed stop”, pg. xxx).

    3. Close the safety door.

    4. Move the motion unit to reference position X=0 mm.

    5. Open the safety door.
