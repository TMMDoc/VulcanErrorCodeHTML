Performing pallet changer function test

Condition * The proximity switch for pallet deceleration/stop has been checked (see “Replacing proximity switch for pallet decelera-tion/stop”, pg. xxx).

  1. Select “Programs” menu in HMI.

  2. Select “Service programs” folder.

  3. Find and run the program “RunPalletLoop.din”. .Fig. 4‑181

    |

  1. Press “Edit” to enter program edit mode.

  2. Modify loop counter and interval time if necessary. .Fig. 4‑182

    |

  1. Press the START button to perform 10 automatic pallet exchanges.

    • The pallet exchange should take place with little sound.

    • The unloaded lift motion should proceed evenly at all 4 corners, both upwards and downward. If this is not the case, check for sluggishness or for grazing against other components.
