Description

Yaskawa: Encoder Communications Error

BM111 cutting unit error 2.36.3216 appears often and sporadically during processing

Cause

Action

Error can be acknowledged only by machine reset. Check the connection between cutting unit and Yaskawa drive

If everything is fine, but error still appears during processing several times, replace encoder cable 2743912

Effect
